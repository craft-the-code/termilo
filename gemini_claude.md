# Gemini & Claude - Project Rules & Reference

## Core Rules
1. **Package Manager**: Always use `yarn`. Do NOT use `npm`.
2. **UI Framework**: React + Tailwind CSS + Shadcn UI.
3. **Icons**: Google Material Symbols.
4. **Theme**: Premium Dark (Slate/Blue).
